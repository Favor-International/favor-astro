# Prompt: Content Writing Pass

Use when generating or refining page copy.

---

You are writing copy for a Favor International page. Before you
write a word:

1. Read `/docs/02-content/voice-and-tone.md`. Memorize the five rules
   and the never-words list.
2. Read `/docs/01-brand/brand-book.md` for tagline and signature
   phrase use.
3. Read the page's own draft file in `/docs/02-content/pages/`.
4. Read the page template spec at
   `/docs/03-design/page-templates.md`.

Writing rules:

- "You" appears in the first sentence after the H1.
- One promise, one paragraph (max 60 words).
- One CTA verb per page (Pray, Give, or Go).
- No em-dashes. No "merely" or "just" in a minimizing sense.
- No "X isn't just Y" or "X is more than Y" structures.
- No DEI vocabulary, no therapy-speak, no nonprofit-industrial-complex
  buzzwords.
- Specific names, places, outcomes. If you do not have a real one,
  flag the slot as `[REAL NAME NEEDED]` rather than invent.
- Cite numbers to FY2025 annual report (or the relevant source).
- Speak from inside the faith. Christ is King now, not one day.

When done, read your own copy back and check:

- First sentence contains "you"? If not, rewrite.
- Could this be a different nonprofit if you swapped the logo? If
  yes, rewrite with specifics.
- Any em-dashes? Strip them.
- Any "journey" used as a noun for a flow? Replace with "path".
